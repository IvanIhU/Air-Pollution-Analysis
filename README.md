# Air Pollution Analysis

This project explores global air pollution data using Python.

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Air pollution dataset containing air quality indicators for cities around the world.

Columns include:
- Country
- City
- AQI Value
- CO AQI Value
- Ozone AQI Value
- NO2 AQI Value
- PM2.5 AQI Value

## Analysis Performed
- Data exploration
- Distribution of pollution values
- Top countries by PM2.5 pollution
- Most polluted cities
- Air quality category distribution
- Correlation analysis between pollutants

## Key Insights
- Several cities reach extremely high pollution levels.
- PM2.5 pollution varies significantly across countries.
- Different pollution indicators show correlations with each other.
