# Global Air Pollution Analysis: Identifying High-Risk Regions

## Overview

This project analyzes global air pollution data to identify regions and cities with the highest pollution levels and to understand patterns across different air quality indicators.

The goal is to provide actionable insights for policymakers, researchers, and organizations.

## Problem Statement

Air pollution is a major global health issue, but pollution levels vary significantly across regions and cities.

This analysis aims to answer:

- Which countries and cities experience the highest pollution levels?
- How do pollution indicators (PM2.5, CO, NO2, Ozone) relate to each other?
- What patterns can be observed in air quality categories?

## Dataset

The dataset contains air quality measurements with the following features:

- Country  
- City  
- AQI Value  
- CO AQI Value  
- Ozone AQI Value  
- NO2 AQI Value  
- PM2.5 AQI Value  

## Tools & Technologies

- Python  
- Pandas  
- Matplotlib  
- Seaborn  

## Key Insights

- Some countries show extremely high average PM2.5 levels, indicating severe air pollution risks  
- Several cities reach AQI values close to 500, representing hazardous air conditions  
- PM2.5 pollution varies significantly across countries, with top regions showing much higher values than low-pollution regions  
- Pollution indicators (CO, NO2, Ozone, PM2.5) show moderate correlations, suggesting interconnected environmental factors  
- Pollution distribution is uneven, with a small number of cities contributing disproportionately to extreme values  

## Recommendations

- Focus environmental policies on high-risk countries with consistently high PM2.5 levels  
- Implement targeted interventions in cities with extreme AQI values  
- Monitor multiple pollutants together due to their correlation  
- Develop early warning systems in regions with rising pollution levels  
- Use data-driven approaches to prioritize environmental investments  

## Conclusion

This analysis highlights significant global disparities in air pollution levels and emphasizes the need for targeted, data-driven environmental strategies.
